# 📚 LangChain Learning Repository

A structured collection of hands-on examples and implementations for learning **LangChain** concepts, including document loading, text splitting, embeddings, vector stores, and retrieval.

> 🚧 This repository is actively being updated with more modules and practical examples.

---

## 📂 Repository Structure

```
.
├── Document_Loaders/       # Load documents from various sources
├── 2_text_splitters/       # Split large documents into manageable chunks
├── 3_Embeddings/           # Generate embeddings using different embedding models
├── 4_vector_stores/        # Store and retrieve embeddings using vector databases
├── 5_retriver/             # Retrieval techniques for RAG applications
└── README.md
```

## 🛠️ Prerequisites

- Python 3.10+
- pip
- Virtual Environment (recommended)

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/<your-username>/<repository-name>.git
cd <repository-name>
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate it:

### Windows

```bash
venv\Scripts\activate
```

### Linux / macOS

```bash
source venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶️ Running Examples

Navigate to any module and execute the Python scripts.

Example:

```bash
cd 3_Embeddings
python embedding_example.py
```
